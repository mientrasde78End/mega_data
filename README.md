# Orbit Portfolio - Backend

# mega_data

Backend unificado para múltiples proyectos:

## Incluye

- Ecommerce (productos, órdenes, pagos Stripe)
- Blog (JWT auth, 1 post por usuario)
- Usuarios compartidos
- Token + JWT authentication
- Swagger / OpenAPI

## Tech

- Django
- Django Rest Framework
- JWT (SimpleJWT)
- Stripe
- SQLite (dev)

## Run local

```bash
python manage.py runserver
```
